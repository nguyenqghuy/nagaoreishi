Installation Instructions
If the file attached to this message is missing or not working please click here to start download.

STEP 1: Download Plugin Files
Unzip and upload all the files in the download package to your root directory where your home or index page is located.
STEP 2: Changes to Existing Files
Standard PHP Sites:
If your home page ends with ".php". Then place the following just before </body> tag on the bottom of your home page, usually the /index.php file.
NOTE: For the best results install on all site pages or in your template footer, so that the plugin code is included on each existing page of your website.
<?php require_once($_SERVER['DOCUMENT_ROOT'] . "/namlinhchinagao.php"); ?>
WordPress Sites:
If your site is built with WordPress or other similar template tools, then place the following code in the FOOTER template. In the WordPress admin panel select Appearance / Editor, then from the Tempaltes list on the left select Footer. Go to the very bottom of the Footer template and add the following code and then "Update File":
<?php require_once($_SERVER['DOCUMENT_ROOT'] . "/namlinhchinagao.php"); ?>
Joomla Sites:
If your site is built with Joomla or other similar template tools, then place the following code on a line of its own at the bottom of the /index.php file:
require_once($_SERVER['DOCUMENT_ROOT'] . "/namlinhchinagao.php");
HTML Sites
If your home page ends with ".htm" or ".html", then place the following just before the </body> tag on the bottom of your home page.
NOTE: For the best results install on each page of your website.

<center><a href="/namlinhchinagao.php?Action=1&k=suc-khoe"> Articles </a> - <a href="/namlinhchinagao.php?Action=2&k=nam-linh-chi"> News </a> - <a href="/namlinhchinagao.php?Action=sitemap"> SiteMap </a> </center>
